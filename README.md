The Contact Book app is a simple program that lets you store and manage people's contact information in a database. Here's what each feature does:
 Add Contact
You enter a person's name, phone number, and email — it saves them permanently to the database so they're still there next time you run the program.
 Search Contact
You type part of a name (even just a few letters) and it finds all matching contacts. For example, searching "ali" would find "Alice", "Malik", "Alicia" etc.
 Delete Contact
You enter the ID number of a contact and it removes them from the database permanently.
 View All Contacts
Shows every contact you've saved, sorted alphabetically by name, with their phone and email.
 Quit
Exits the program.

In order to try it out place them all in one folder and run contacts_main
